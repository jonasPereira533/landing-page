# 🐧 Linux Landing Page

Landing page desenvolvida em HTML e CSS puros, contando a história do Linux de forma visual e resumida: das origens com Linus Torvalds até os dias atuais, passando por curiosidades e marcos importantes do sistema operacional.

##  Preview

>

##  Funcionalidades

- Linha do tempo com os principais marcos da história do Linux
- Seção de curiosidades sobre o sistema e sua comunidade
- Ilustração autoral do Tux (mascote do Linux) em estilo geométrico, feita em SVG
- Layout responsivo

##  Tecnologias

- HTML5
- CSS3
- SVG (ilustrações e ícones)


##  Como rodar localmente

```bash
git clone https://github.com/jonasPereira533/landing-page.git
cd landing-page
```

Depois é só abrir o `index.html` no navegador (ou usar a extensão *Live Server*, se estiver no VS Code).
